Qué puede hacer

Menos búsqueda. Más trabajo terminado.

Navegar carpetas, encontrar archivos, leer Docs, consultar o actualizar Sheets, descargar, subir, copiar, mover y renombrar.

drive_list_folder({ folderId })
sheets_read({ ranges: ["Ventas!A1:F50"] })
drive_upload_file({ source, parentId })

// Papelera exige confirmación explícita
drive_trash_file({ fileId, confirm: true })
